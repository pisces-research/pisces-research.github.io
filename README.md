# pisces-research.github.io
The Pisces Research Limited website, deployed on GitHub pages.

The site is built using [quarto](https://quarto.org/), and deployed to GitHub pages by running the `quarto publish gh-pages` command, following the second approach detailed [here](https://quarto.org/docs/publishing/github-pages.html).

The homepage animations are created using the [GSAP](https://gsap.com/) framework.
